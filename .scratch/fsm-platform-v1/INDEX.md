# FSM Platform v1 — Issue Index

Tracer-bullet vertical slices for the greenfield FSM GPS Field Service Management platform
(NestJS backend + React/Vite admin dashboard + RN/Expo SE mobile app). Each issue is a thin
end-to-end slice (schema → API → UI → tests). Generated by `/to-issues` from `docs/`
(priority: PRD-fsm-admin-dashboard.md → fsm-business-technical-workflow.md → backend design docs).

Triage: AFK slices are `ready-for-agent`; HITL slices (01, 03) are `ready-for-human`.
Sequence loosely follows the backend LLD phases P0–P7. "Blocked by" gives the build order.
